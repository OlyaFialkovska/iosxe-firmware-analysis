The original IOS-XE package files are very large (hundreds of MB each) and are not included in this repository due to GitHub size limits.

Example packages discovered in the firmware:

* cat9k-rpbase.16.12.07.SPA.pkg (~512 MB)
* cat9k-webui.16.12.07.SPA.pkg
* cat9k-sipbase.16.12.07.SPA.pkg

These packages were extracted from the SquashFS filesystem and analyzed during the firmware investigation, but are not stored in the repository.
