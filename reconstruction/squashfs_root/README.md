## SquashFS Root Filesystem

During the extraction of the SquashFS filesystem, several IOS-XE package files were discovered, for example:

* cat9k-rpbase.16.12.07.SPA.pkg
* cat9k-webui.16.12.07.SPA.pkg
* cat9k-sipbase.16.12.07.SPA.pkg

* cat9k-rpbase.16.12.06.SPA.pkg
* cat9k-webui.16.12.06.SPA.pkg
* cat9k-sipbase.16.12.06.SPA.pkg

These packages are quite large (hundreds of MB each), so they are not included in this repository due to GitHub size limits.

They were extracted from the SquashFS filesystem and analyzed during the firmware investigation.
